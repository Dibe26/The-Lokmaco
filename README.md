# 🥞 The Lokmaco — Онлайн-заказ вафель и панкейков

Учебный проект по дисциплине **Software Engineering**.  
Данный веб-сервис предназначен для автоматизации процесса выбора и заказа десертов в кафе "The Lokmaco".

---

## 👥 Наша Команда (Scrum Team)
Мы распределили роли согласно методологии Scrum для эффективного управления разработкой:

*   **Диёра** — *Product Owner (Владелец продукта)*: Управление требованиями, написание Технического задания (ТЗ) и контроль соответствия проекта целям бизнеса.
*   **Данил** — *Scrum Master / Lead Developer*: Ответственный за архитектуру кода, процесс разработки и развертывание проекта на GitHub Pages.
*   **Виолета** — *UML Architect / Developer*: Проектирование структуры системы, создание UML-диаграмм (Object Diagram) и разработка логики.

---

## 🛠 Технологический стек
Для реализации проекта были использованы следующие инструменты:
*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS).
*   **Моделирование:** StarUML (для создания UML-диаграмм в формате `.mdj`).
*   **Управление:** GitHub Projects (Scrum-доска).
*   **Хостинг:** GitHub Pages.

---

## 🚀 Методология разработки (Scrum)
Проект реализован по гибкой методологии Scrum:
*   **Спринты:** Разработка была разбита на итерации (спринты) длительностью в 1 неделю.
*   **Артефакты:** 
    *   *Product Backlog:* На основе ТЗ сформирован список задач.
    *   *UML Model:* Описана структура объектов системы (заказы, меню, корзина).
    *   *Increment:* Работающий веб-интерфейс с возможностью выбора товаров.
